## 자손 선택자 vs 자식 선택자
```css
/* 자손 선택자 */
div ul
/* 자식 선택자 */
div > ul
```
-|자손|자식|
:-:|:-:|:-:
영향|div 아래 모든 ul|div 바로 아래 있는 ul만
쓰임|div 태그<br>ul태그(영향 O)<br>ul태그(영향 O)|div 태그<br>ul태그(영향 O)<br>ul태그(영향 X)
