> **인접(Adjacency)**  
두 개의 정점에 간선이 존재(연결)

## 인접 행렬
 - V*V 크기의 2차원 배열을 이용해서 간선 정보를 저장.  
 - 배열의 배열(포인터 배열)
```java
static int[][] graph = new int[V][V]; //[노드][인접 정보]
```
노드가 적고 간선이 많을 때 사용하라!

> 노드가 많고 간선이 적을 땐, **리스트**로...
